SkillSwap AI - نسخه دمو فارسی (Static Demo)
فایل‌ها:
- index.html (صفحه اصلی)
- signup.html (فرم ثبت‌نام و ذخیره محلی)
- profile.html (نمایش پروفایل ذخیره شده و دانلود JSON)
- styles.css (استایل‌ها)
- scripts.js (منطق فرم، localStorage و دانلود JSON)

نحوه اجرا:
1) فایل skillswap_demo_fa.zip را استخراج کن.
2) فایل index.html را در مرورگر باز کن.
3) از منوی ثبت‌نام، اطلاعات را وارد کن و ذخیره کن.
4) به صفحه پروفایل برو تا داده‌ها نمایش داده شوند و فایل JSON را دانلود کنی.

توضیح فنی:
- این نسخه دمو از localStorage مرورگر برای ذخیره پروفایل استفاده می‌کند (بدون سرور).
- برای نسخه واقعی باید یک backend با دیتابیس (مثلاً Firebase یا PostgreSQL) راه‌اندازی شود.
